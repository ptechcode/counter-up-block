A block to count up to display a number in specific time.

== Description ==

Gutenberg count up to display number.

## FEATURES ##
 - Choosing a number to count up
 - Easily assign the number to count up within certain time
 - Allow user to customize the number with difference font and size


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/counter-up-block` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==

Yes, it will work with any standard WordPress theme.
